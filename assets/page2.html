Tips

Back
  • 1. Formula should be defined in such form "c = a + b" or "a + b" where c is the answer we need to find out & "a+b" is actual formula
  • 2. The answer display is the calculation of the last formula displayed in the text form.
  • Ex1:   
    s = a + b
    b = 4
    a = b + 1
    Ans: a = 5
  • Ex2:   
    a = b + 1
    b = 4
    s = a + b
    Ans: s = 9
  • 3. Embedding your own function: Suppose you have a function name "funcA" and you want to apply this function in another calculation for "funcB" without redefine the content of "funcA". xCalc can help you! Simply, add the prefix "my_" before "funcA", it will automatically embed funcA's content in the new formula.
  • Ex3:   
    Predefine in another formula: funcA = a * b
    In the new formula,
    funcB = c + my_funcA
    Then, the actual formula we're calculating is:
    funcB = c + a * b
  • 4. Create new formula: name and formula fields can't be empty, description field can be either ways. When description field of formula is empty, it will display "n/a" on the message box when you're trying to look for the description
  • 5. The "back" button leads you back to the previous page
